#### 面向对象

~~~python
# 面向对象 三大特性
   封装
		- 对同一类方法和属性封装到类中
    	- 将数据封装到对象中
        eg. drf中对原生request进行了封装
   继承 
        - 多个类中相同的方法和属性只实现一次(提取到基类中)，通过继承的关系来实现
    	eg. drf中的视图的编写时继承 APIView
        多继承： 新式类 旧式类
   多态
		- 鸭子模型
~~~

* 鸭子模型

  ~~~python
  class WX:
      def send():
          pass
      
  class Email:
      def send():
          pass
      
  class Msg:
      def send():
          pass
  
  def func(arg):
      arg.send()
      
  q = WX()
  func(q)
  
  q = Email()
  func(q)
  
  ...
  ~~~
